 EkoStat LIVE
Real-time economic dashboard for Indonesia.

🌐 Live: https://ekostatslivekyy.netlify.app
Built with: Vanilla JS, CoinGecko API, Frankfurter API, Alternative.me

 Features
- Live forex rates (USD/IDR, EUR, SGD...)
- XAU, BTC, ETH spot prices
- Fear & Greed Index
- Indonesian macro indicators
